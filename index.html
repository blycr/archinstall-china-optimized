⚠️ 重要提示
直接运行 archinstall 命令会使用系统官方原版，不包含中国优化！
必须使用 python -m archinstall 从本仓库运行才能启用优化。

快速开始

# 1. 克隆此仓库 git clone https://github.com/blycr/archinstall-china-optimized.git # 2. 进入目录 cd archinstall-china-optimized # 3. 运行优化版安装器（必须使用 python -m 方式） python -m archinstall
